Руководство для Графиков Отпусков

Основная схема работы с Графиками Отпусков в системе следующая:

  1. Вначале сформировать в системе новый график отпусков для своего подразделения на предстоящий год. График отпусков формируется на основе данных из учётной системы «Парус» без дат и продолжительности отпусков.

  2. Затем проставить в сформированном графике отпусков даты и продолжительность отпусков для каждого сотрудника подразделения.

  3. Распечатать график отпусков.

Как создать новый график отпусков.

  1. Перейти на вкладку “Мои Графики Отпусков”. На экране появится список графиков отпусков.

  2. На экране, мышью, выбрать нужное подразделение и нажать кнопку «Создать Новый График». На экране появится форма для ввода параметров формирования графика отпусков.

  3. В форме ввода параметров указать год нового графика отпусков.

  4. На экране, мышью, нажать кнопку «Создать». Система создаст новый график отпусков на основе данных системы «Парус» и вернётся к списку графиков отпусков. Теперь созданный график можно просмотреть, напечатать, изменить, удалить.


Как просмотреть и изменить данные графика отпусков.

  1. Перейти на вкладку “Мои Графики Отпусков”. Выбрать нужное подразделение. На экране появится список графиков отпусков.

  2. Найти в списке нужный график и нажать мышью на символ редактирования записи. Откроется список строк выбранного графика отпусков. В списке для каждого сотрудника отображаются даты и продолжительность их отпусков.

  3. Выберите в списке строку данные которой нужно изменить. Нажмите мышью на символ редактирования записи или на имя сотрудника в выбранной строке. Откроется форма с учётными данными для выбранной строки графика.

  4. Для того, чтобы ввести данные отпуска для сотрудника, нажмите мышью на экране кнопку «Создать Новую Запись». Укажите параметры отпуска: продолжительность в днях и дату начала отпуска. Нажмите мышью на экране кнопку «Добавить». Если данные отпуска были указаны верно, то на экране, в списке отпусков, появится новая строка.

  5. Если требуется удалить строку отпуска нажмите мышью на экране на символ удаления (крестик) в нужной строке. Строка исчезнет из списка.

  6. Для возврата к списку сотрудников нажмите кнопку «Закрыть». На экран вернётся список строк графика отпусков с перечнем сотрудников подразделения. Теперь можно выбрать следующую строку и продолжить изменения данных графика.

  7. Когда произведены все изменения в списке строк графика нажать мышью на кнопку «Закрыть». На экран вернётся список графиков отпусков подразделения.



Как напечатать график отпусков.

  1. Перейти на вкладку “Мои Графика Отпусков”. Выбрать нужное подразделение. На экране появится список графиков отпусков.

  2. Найти в списке нужный график и нажать мышью на символ редактирования записи. Откроется список строк выбранного графика отпусков.

  3. На экране, мышью, нажмите кнопку «Напечатать График». Откроется окно с данными отчётной формы графика отпусков.

  4. В заголовке, в параметрах формы, выбрать из списка вид отчета «RTF» и нажать мышью на кнопку «Экспорт». Откроется окно с параметрами открытия документа.

  5. Выбрать «Открыть в », мышью нажать кнопку «ОК». Откроется приложение Microsoft Word с печатной формой документа. Здесь можно ещё раз просмотреть документ и внести исправления. Внимание: исправления, внесённые в документ на данном этапе не будут сохранены в базе данных, т.е. при повторной печати необходимо вновь вносить изменения в документ.

  6. Отослать документ на печатающее устройство средствами приложения Microsoft Word.



Как удалить график отпусков.

  1. Перейти на закладку “Мои Графика Отпусков”. Выбрать нужное подразделение. На экране появится список графиков.

  2. Найти в списке нужный график и нажать мышью на символ редактирования записи. Откроется список строк выбранного графика.

  3. На экране, мышью, нажать кнопку «Удалить». Система удалит выбранный график и вернётся к списку графиков отпусков.
