Руководство Табельщика

Основная схема работы Табельщика в системе следующая:

  1. Вначале формирует в системе табель для своего подразделения за очередной учётный месяц. Табель формируется на основе данных из учётной системы «Парус».

  2. Затем проверяет сформированный табель и при необходимости вносит в него изменения. В результате внесения изменений в табель появляются расхождения между данными табеля и системы «Парус».

  3. Далее проверяет (печатает) перечень расхождений и при не обходимости, совместно с отделом кадров и бухгалтерией, участвует в устранении расхождений.

  4. По готовности передаёт табель в бухгалтерию для расчётов.

  5. При необходимость формирует корректирующий табель.

Как создать табель.

  1. Перейти на вкладку“Мои Табели”. На экране появится список табелей.

  2. На экране, мышью, нажать кнопку «Создать новый табель». На экране появится форма для ввода параметров формирования табеля.

  3. В форме ввода параметров указать год и месяц нового табеля.

  4. На экране, мышью, нажать кнопку «Создать». Система создаст новый табель на основе данных системы Парус и вернётся к списку табелей. Теперь созданный табель можно просмотреть, напечатать, изменить.


Как просмотреть и изменить данные табеля.

  1. Перейти на вкладку “Мои Табели”. На экране появится список табелей.

  2. Найти в списке нужный табель и нажать мышью на символ редактирования записи либо на название месяца. Откроется список строк выбранного табеля с данными за каждый день месяца. В списке для каждого сотрудника и для каждого дня месяца отображается тип дня и общее количество отработанных часов за день.

  3. Выберите в списке строку табеля данные которой нужно изменить. Нажать мышью на имя сотрудника в выбранной строке. Откроется форма с учётными данными для выбранной строки табеля.

  4. Здесь можно изменить данные выбранной строки. Для каждого дня месяца можно изменить тип дня, количество отработанных часов.

  5. На экране, мышью, нажать кнопку «Сохранить». Система запишет изменёния в базу данных.

  6. На экране, мышью, нажать кнопку «Закрыть». На экран вернётся список строк табеля. Теперь можно выбрать следующую строку и продолжить изменения данных табеля.

  7. Когда произведены все изменения в списке строк табеля нажать мышью на кнопку «Закрыть». На экран вернётся список табелей подразделения.



Как напечатать табель.

  1. Перейти на вкладку “Мои Табели”. На экране появится список табелей.

  2. Найти в списке нужный табель и нажать мышью на символ редактирования записи либо на название месяца. Откроется список строк выбранного табеля с данными за каждый день месяца. В списке для каждого сотрудника и для каждого дня месяца отображается тип дня и общее количество отработанных часов за день.

  3. На экране, мышью, нажать кнопку «Напечатать табель». Откроется окно с данными отчётной формы табеля.

  4. В заголовке, в параметрах формы, выбрать из списка вид отчета «RTF» и нажать мышью на кнопку «Экспорт». Откроется окно с параметрами открытия документа.

  5. Выбрать «Открыть в », мышью нажать кнопку «ОК». Откроется приложение Microsoft Word с печатной формой документа. Здесь можно ещё раз просмотреть документ и внести исправления. Внимание: исправления, внесённые в документ на данном этапе не будут сохранены в базе данных, т.е. при повторной печати необходимо вновь вносить изменения в документ.

  6. Отослать документ на печатающее устройство средствами приложения Microsoft Word.



Как удалить табель.

  1. Перейти на закладку “Мои Табели”. На экране появится список табелей.

  2. Найти в списке нужный табель и нажать мышью на символ редактирования записи либо на название месяца. Откроется список строк выбранного табеля с данными за каждый день месяца. В списке для каждого сотрудника и для каждого дня месяца отображается тип дня и общее количество отработанных часов за день.

  3. На экране, мышью, нажать кнопку «Удалить». Система удалит выбранный табель и вернётся к списку табелей.


Что такое расхождения.

Термин «Расхождения» используется в системе для обозначения факта отличия информации в табеле от информации в системе Парус. Первоначально, сразу после создания табеля, информация в табеле и в системе Парус полностью совпадает, расхождений нет. В дальнейшем в процессе корректировки либо данных табеля либо данных в системе Парус появляются различия в данных, т.е. появляются расхождения. Причины расхождения в данных могут носить как объективный характер, так и быть следствием ошибок учёта. На практике важно оперативно выявлять и устанять расхождения в учёте. В данной системе табельного учёта имеются средства для выявления и документирования расхождений.



Как обнаружить факт наличия расхождений.

  • В списке табелей на закладке “Мои Табели” табель с рахождениями помечается системой символом «+» красного цвета.

  • На странице табеля, в котором есть расхождения, появляется кнопка «Расхождения» красного цвета.

  • На странице спецификации табеля, в которой есть расхождения, появляется кнопка «Расхождения» красного цвета.



Как увидеть перечень расхождений

На странице табеля, в котором есть расхождения, мышью на жмите кнопку «Расхождения». Откроется окно с перечнем расхождения для данного табеля. Этот перечень расхождения можно напечатать на принтере.


Как распечатать перечень расхождений табеля

  1. Перейти на вкладку “Мои Табели”. На экране появится список табелей.

  2. Найти в списке нужный табель и нажать мышью на символ редактирования записи либо на название месяца. Откроется список строк выбранного табеля с данными за каждый день месяца. Если в этом табеле есть расхождения, то на экране появится кнопка «Расхождения».

  3. На экране, мышью нажмите кнопку «Расхождения». Откроется окно с перечнем расхождений для данного табеля.

  4. В заголовке, в параметрах формы, выбрать из списка вид отчета «RTF» и нажать мышью на кнопку «Экспорт». Откроется окно с параметрами открытия документа.

  5. Выбрать «Открыть в », мышью нажать кнопку «ОК». Откроется приложение Microsoft Word с печатной формой документа.

  6. Отослать документ на печатающее устройство средствами приложения Microsoft Word.



Как передать табель в бухгалтерию для расчётов.

  1. Перейти на вкладку “Мои Табели”. На экране появится список табелей.

  2. Найти в списке нужный табель и нажать мышью на символ редактирования записи либо на название месяца. Откроется список строк выбранного табеля с данными за каждый день месяца. Если табель не передан для расчёта в бухгалтерию, то на экране будет видна кнопка «Передать для расчёта» и в заголовке страницы будет фраза «Не передан. Не принят.». Если такой кнопки на экране нет, то это может означать, что табель уже был ранее передан для расчёта, в этом случае в заголовке страницы будет фраза «Передан (дата передачи)» (Табели, переданные для расчёта в бухгалтерию, можно отозвать для исправлений при условии, что этот табель ещё не принят к учёту в бухгалтерии).

  3. На экране, мышью, нажать кнопку «Передать для расчёта». После этого кнопка «Передать для расчёта» исчезнет с экрана и в заголовке табеля появится фраза «Передан (дата передачи). Не принят.».


Как отозвать табель из бухгалтерии для исправления.

  1. Перейти на вкладку “Мои Табели”. На экране появится список табелей.

  2. Найти в списке нужный табель и нажать мышью на символ редактирования записи либо на название месяца. Откроется список строк выбранного табеля с данными за каждый день месяца. Если табель передан для расчёта в бухгалтерию но ещё не принят к учёту в бухгалтерии, то на экране будет видна кнопка «Отозвать» и в заголовке страницы будет фраза «Передан (дата передачи). Не принят.». Если такой кнопки на экране нет, то это может означать, что либо табель ещё не был передан для расчёта (в этом случае в заголовке страницы будет фраза «Не передан. Не принят.») либо табель уже был принят в бухгалтерии к учёту (в этом случае в заголовке страницы будет фраза «Передан (дата передачи). Принят (дата принятия).») Если табель уже принят в бухгалтерии к учёту, то отзыв табеля возможен только в том случае, если бухгалтер вначале снимет с табеля отметку о принятии к учёту.

  3. На экране, мышью, нажать кнопку «Отозвать». После этого кнопка «Отозвать» исчезнет с экрана и в заголовке табеля появится фраза «Не передан. Не принят.».

Как создать корректирующий табель.

  1. Перейти на вкладку “Мои Табели”. На экране появится список табелей.

  2. Найти в списке нужный табель и нажать мышью на символ редактирования записи либо на название месяца. Откроется список строк выбранного табеля с данными за каждый день месяца.

  3. На экране, мышью, нажать кнопку «Создать Корректирующий Табель». На экране появится форма с параметрами формирования корректирующего табеля. Эти параметры устанавливаются автоматически и показываются для контроля.

  4. В созданном корректирующем табеле можно удалить ненужные строки. Для этого мышью, нажать кнопку «Изменить, в первой графе отметить ненужные строки и нажать кнопку «Удалить отмеченные строки.

  5. Теперь созданный корректирующий табель можно просмотреть, напечатать, изменить. Печать и изменение табеля выполняются также, как и с основным табелем.
